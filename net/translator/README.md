# Translator
A tool that can translate text.

## Usage
```

Usage: translator.exe command [Options] <Text>

The commands are:
        translate
        detect
        getlangs

Options:
  -c string
        translation service provider: yandex (default "yandex")
  -f string
        response format (default "json")
  -k string
        api key for translation service
  -s string
        source language (default "zh")
  -t string
        target language (default "en")

```
